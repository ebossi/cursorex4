# 나의 포트폴리오 웹사이트

이 웹사이트는 HTML5, CSS, JavaScript를 사용하여 만든 개인 포트폴리오 웹사이트입니다.

## 기능

- 반응형 디자인
- 스무스 스크롤 네비게이션
- 프로젝트 갤러리
- 기술 스택 표시
- 연락처 폼

## 설치 방법

1. 저장소를 클론합니다:
```bash
git clone [your-repository-url]
```

2. 프로젝트 디렉토리로 이동합니다:
```bash
cd portfolio-website
```

3. 웹사이트를 실행합니다:
- 로컬 서버를 사용하거나
- VS Code의 Live Server 확장 프로그램을 사용하거나
- 브라우저에서 직접 `index.html` 파일을 열어볼 수 있습니다.

## 사용된 기술

- HTML5
- CSS3
- JavaScript
- Font Awesome (아이콘)
- Google Fonts
